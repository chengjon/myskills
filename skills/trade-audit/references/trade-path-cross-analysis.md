## 方法论

### Wilson Score Interval (95% CI)

二项比例的精确置信区间。当样本量小时，Wilson CI 比正态近似更稳定。

```python
def wilson_ci(wins, total, z=1.96):
    n = total
    p_hat = wins / n
    denom = 1 + z*z / n
    center = (p_hat + z*z / (2*n)) / denom
    spread = z * math.sqrt((p_hat*(1-p_hat) + z*z/(4*n)) / n) / denom
    return (center - spread) * 100, (center + spread) * 100
```

n<30 的行标注 ⚠️样本不足，不单独解读。

### Cohen's d 效应量

| |d| | 解读 |
|------|------|
| <0.2 | 忽略 |
| 0.2-0.5 | 小效应 |
| 0.5-0.8 | 中效应 |
| >0.8 | 大效应 |

### 行为改善指数

```
BII = (胜率变化 + 冲动率变化(负=改善) + 持仓天数变化) / 3
>0 改善, <0 恶化
```

## 关键发现 (2026-06-04 首次运行)

### 1. BOLL位置是唯一正期望区分因子

| BOLL区间 | 超短(≤3d) | 短线(4-7d) | 短中(8-20d) | 中线(21d+) |
|----------|----------|-----------|------------|-----------|
| ≤20% | 60% / +491 | 57% / +398 | 60% / +475 | 54% / -337 |
| 20-80% | 38% / -35 | 46% / -47 | 55% / -552 | 67% / -260 |
| >80% | 32% / -551 | 32% / -1305 | 43% / -340 | 13% / -1139 |

**结论**: BOLL≤20% 是唯一正期望区间(胜率52-67%, 笔均+398~+491)。BOLL>80% 全线亏损，尤其超短+高BOLL最致命(笔均-1305)。

### 2. 全局连亏是最强行为预测因子

| 连亏次数 | 胜率 | 笔均盈亏 | Cohen's d vs 0次 |
|---------|------|---------|-----------------|
| 0次 | 57-65% | +583 | — |
| 1次 | 33-47% | -310 | -0.36 (小) |
| 2次 | 29-46% | -780 | -0.50 (中) |
| 3次+ | 20-30% | -1143 | -0.64 (中大) |

**结论**: 连亏3次+后胜率骤降至20-30%，效应量0.64(中大)。这是最可操作的行为信号——连亏后应强制暂停交易。
